## 线下测试代码

### 实现功能

* 能够生成1000张随机地图

* 能够判断每张地图是否配送成功

* 能够计算平均步数

* 能够计算总的耗时

### 使用方法

1. 首先在`test_mask.py`文件中修改变量`mask_program`为你要测试的配送算法程序路径；

2. 随后在命令行下执行`python3 test_mask.py`即可开始线下测试。
